# 五子棋规则

## 基本规则

### 棋盘设置
- 使用 15×15 的棋盘，横竖各 15 条线，形成 225 个交叉点
- 双方分别使用黑白两色的棋子
- 通常黑棋先行，白棋后行

### 游戏玩法
1. 双方轮流在棋盘的交叉点上放置棋子
2. 棋子一旦放下，不能移动或拿走
3. 先将己方的五个棋子在横、竖、斜方向上连成一线的一方获胜

### 获胜条件
- 五子连珠：在横、竖、斜任意方向上，有五个同色棋子连续排列
- 只要形成五子连珠，游戏立即结束，该方获胜

## 禁手规则（专业规则）

### 黑棋禁手
在专业比赛中，为平衡黑棋先行的优势，黑棋有以下禁手：
1. **三三禁手**：同时形成两个活三
2. **四四禁手**：同时形成两个四
3. **长连禁手**：形成六子或以上的连珠

### 活三与冲四
- **活三**：在棋盘某一条线上有同色 3 子连续，且两端都有空位
- **冲四**：除活四外的其他四子连线

## 基本术语

### 连线类型
- **活四**：四子连续且两端都有空位
- **冲四**：四子连续但只有一端有空位
- **活三**：三子连续且两端都有空位
- **眠三**：三子连续但只有一端有空位
- **活二**：两子连续且两端都有空位
- **眠二**：两子连续但只有一端有空位

### 棋形术语
- **双三**：同时形成两个活三
- **双四**：同时形成两个四
- **连珠**：五子或以上连续排列

## 比赛规则

### 时间限制
- 每位棋手有固定的思考时间
- 超时判负
- 可设置读秒时间

### 开局规则
- **三手交换**：前两手棋自由下，第三手后对手可选择交换颜色
- **五手两打**：第五手时，黑方需要在两个不同位置各下一子，白方选择其中一个

### 和棋条件
- 双方同意和棋
- 棋盘下满无人获胜
- 出现重复局面三次以上

## 变种玩法

### 连珠（Renju）
- 国际标准规则，黑棋有禁手限制
- 更严格的平衡规则
- 需要专业的开局规则

### 自由五子棋
- 无禁手限制
- 更简单直接
- 适合初学者

### 六子棋
- 需要六子连珠才能获胜
- 游戏时间更长，策略更复杂

## 基本策略

### 进攻策略
1. **建立双威胁**：同时制造多个进攻路线
2. **设置陷阱**：故意露出破绽引诱对手
3. **连续进攻**：保持压力不让对手喘息

### 防守策略
1. **及时防守**：对手形成威胁时立即阻挡
2. **主动防守**：在对手形成威胁前预先防守
3. **反守为攻**：在防守的同时寻找反击机会

### 中盘技巧
1. **控制中心**：占据棋盘中央位置
2. **灵活机动**：保持棋子的灵活性
3. **平衡发展**：在多个方向同时发展

## 常见错误

### 初学者常见错误
1. **忽视防守**：只顾进攻忘记防守
2. **急于求成**：过早暴露进攻意图
3. **缺乏远见**：只看眼前不看长远
4. **忽视中盘**：开局和残盘都很重要

### 高级玩家注意
1. **避免简单陷阱**：小心常见的战术陷阱
2. **保持冷静**：不要被对手的假动作迷惑
3. **时间管理**：合理分配思考时间
4. **心理战术**：保持良好的心态

## 训练建议

### 基础训练
1. **熟记棋形**：掌握各种基本棋形
2. **练习解题**：多做五子棋题目
3. **复盘分析**：分析自己的对局
4. **学习经典**：研究经典对局

### 提高方法
1. **参加比赛**：通过实战提高水平
2. **使用软件**：借助训练软件练习
3. **向高手学习**：观摩高水平对局
4. **总结经验**：不断完善自己的战术体系
